## 📋 项目背景

### 我们是谁
我们是一个**AI服装SaaS平台**，服务于服装电商行业（淘宝/天猫/抖音/小红书/跨境电商的商家）。

### 核心技术栈
- **后端：** Express.js + Knex.js + MySQL 8 + Redis + BullMQ
- **前端：** Next.js 14 + Ant Design 5
- **AI推理平台：** RunningHub（将ComfyUI工作流转为API，按调用时长收费¥0.005/秒）
- **存储：** 腾讯云COS对象存储

### RunningHub平台简介
RunningHub是一个专门的AI推理平台，核心特性：
1. **工作流转API**：上传ComfyUI工作流JSON，自动生成RESTful API
2. **按需付费**：按API调用时长计费（¥0.005/秒），无基础费用
3. **弹性扩容**：自动扩缩容，支持高并发
4. **模型库丰富**：预装1000+开源模型（HuggingFace同步）
5. **全球加速**：CDN加速，推理速度快

**为什么选择RunningHub而不是自建GPU？**
- ✅ 零GPU服务器成本（自建4×RTX 4090需要¥8,000/月）
- ✅ 零运维成本（不需要管理服务器、模型、环境）
- ✅ 成本可控（按实际使用付费，闲时零成本）
