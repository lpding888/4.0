## 🚀 使用流程（3步走）

### 第1步：提问GPT-5 Pro

1. 打开 `01-问题文档/` 文件夹
2. 选择你要问的问题（如：`问题00-CMS配置系统深度技术分析.md`）
3. 打开文件，找到"完整提问文本（复制即用）"部分
4. 复制整段文本（从 \`\`\` 开始到 \`\`\` 结束）
5. 打开 ChatGPT Pro，确保使用 **GPT-5 Pro** 模型
6. 粘贴提问，点击发送
7. 等待 5-15 分钟（模型深度思考中...）

### 第2步：保存答案

1. GPT-5 Pro回答完成后，复制完整答案
2. 打开 `02-GPT5Pro答案/` 文件夹
3. 找到对应的答案文件（如：`问题00-CMS系统-GPT5Pro答案.md`）
4. 打开文件，填写：
   - 提问时间
   - 思考时间
   - 核心洞察（3-5条最关键的发现）
   - 完整答案（粘贴GPT-5 Pro的回答）
5. 保存文件

### 第3步：整理行动清单

1. 阅读GPT-5 Pro的答案
2. 提取可执行的行动项
3. 在 `03-行动清单/` 文件夹创建对应文件
4. 按优先级整理：
   - 立即行动（1周内）
   - 短期行动（1个月内）
   - 中期行动（3个月内）
5. 分配负责人和截止日期
6. 定期复盘，更新进度

## 💡 使用技巧

### 如何高效提问？
1. **提供完整上下文**：不要删减问题文档中的背景信息
2. **一次一个问题**：不要同时问多个问题，GPT-5 Pro需要深度思考
3. **给足思考时间**：不要催促，等待5-15分钟
4. **明确要求输出**：问题文档已包含明确的输出要求，不需要修改

### 如何整理答案？
1. **先看核心洞察**：GPT-5 Pro的答案很长（8000-15000字），先提取核心观点
2. **标记重点**：用高亮、加粗标记关键信息
3. **转化为行动**：把建议转化为可执行的任务清单
4. **与团队讨论**：不要独自决策，与团队一起讨论可行性

### 如何执行行动清单？
1. **设定优先级**：P0（必须做）、P1（应该做）、P2（可以做）
2. **分配负责人**：明确每个任务的负责人
3. **设置截止日期**：用截止日期推动执行
4. **定期复盘**：每周/每月回顾进度，调整计划
