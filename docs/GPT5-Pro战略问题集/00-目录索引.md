## 🎯 使用指南

### 如何使用这份文档？

1. **按需选择问题**
   - 根据当前项目阶段选择对应问题
   - Phase 1（生存期）→ Phase 2（增长期）→ Phase 3（成熟期）
   - 不要一次性问完所有问题，分阶段提问

2. **提问步骤**
   - 打开对应问题的独立文档（在 `01-问题文档/` 文件夹）
   - 复制"完整提问文本"
   - 打开ChatGPT Pro（确保使用GPT-5 Pro模型）
   - 粘贴提问，点击发送
   - 等待5-15分钟（模型深度思考）
   - 保存答案到 `02-GPT5Pro答案/` 文件夹

3. **答案应用**
   - 每个问题下都有"如何应用"指南
   - 将GPT-5 Pro的答案粘贴到对应的答案文件
   - 整理行动清单到 `03-行动清单/` 文件夹
   - 制定实施计划并执行

4. **预期时间投入**
   - 每个问题提问：5分钟
   - GPT-5 Pro思考：5-15分钟
   - 阅读答案：20-40分钟
   - 整理行动计划：30-90分钟
   - **总计：1-2.5小时/问题**
