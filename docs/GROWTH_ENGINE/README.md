## 📊 项目概述

### 两大系统

#### 🎁 活动营销系统
- **核心功能**：优惠券管理、活动创建、折扣计算、核销统计
- **预期效果**：新用户转化率+15%，老用户复购率+10%
- **预计工期**：8-11天

#### 🚀 分销代理系统
- **核心功能**：推荐关系、佣金结算、提现管理、数据统计
- **预期效果**：CPA成本-30%，1个月推广500+新用户
- **预计工期**：11-14天

### 总体时间线
**19-25个工作日**（约4-5周）

---

## ⚠️ 关键风险

### 🔥 高风险（必须重视）
1. **并发超发** - 必须使用行锁+事务
2. **财务安全** - 所有金额后端计算
3. **自推自买** - 监控异常推广行为

### ⏰ 时间风险
- 涉及前后端多个模块
- 已预留5天Buffer
- 建议后端先行，前后端并行
